# Projet React + Bootstrap — CEF

## Prérequis

Avant de lancer le projet, assurez-vous d’avoir :

Node.js (version 18 ou supérieure)

npm ou yarn

Navigateur web moderne (Chrome, Firefox, Edge…)


## Installation et récupération du projet

### Cloner le dépôt GitHub :

git clone https://github.com/ton-utilisateur/nom-du-projet.git
cd nom-du-projet


### Installer les dépendances :

npm install

### ou avec yarn

yarn install

## Lancement du projet

Pour lancer l’application en mode développement :

npm start
ou 
yarn start


L’application sera accessible dans votre navigateur à l’adresse : http://localhost:3000

Toute modification du code sera automatiquement reflétée grâce au hot reload de React.

Pour créer une version de production (optionnel) :

npm run build
ou 
yarn build
